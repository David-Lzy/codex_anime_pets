# Assistant-004 Creation Prompt / Assistant-004 创作题词

## Original Creative Brief / 原始创作简述

Pet name: Assistant-004

Create an original, copyright-safe Codex desktop pet inspired by the archetype of a sharp red-haired lab assistant from a retro Japanese sci-fi visual novel.

Do not copy any existing anime character exactly. Do not reproduce a named character one-to-one. Instead, create an original chibi "Lab Assistant No.004" character with a similar mood: genius researcher, skeptical assistant, worldline observer, and sarcastic coding partner.

Core concept:

A tiny chibi AI research assistant for an AI engineer. She helps watch over code agents, model training, servers, debugging tasks, experiments, and long-running infrastructure jobs. She feels like a brilliant neuroscience / computer science lab member who is rational, impatient with nonsense, but secretly reliable.

Visual direction:

- original anime-inspired chibi assistant
- long reddish-brown / chestnut hair, clean recognizable silhouette
- sharp blue or violet eyes, calm but slightly judging expression
- white short lab-coat-like jacket or scientist jacket
- simple collared shirt with a small red tie or dark red accent
- dark lower outfit, simple boots or stockings
- subtle retro science-fiction laboratory feeling
- optional tiny green/teal terminal glow or worldline-style light accent
- no readable text, no logos, no exact symbols from any franchise
- no speech bubbles, no background, no UI panels
- transparent-background friendly
- readable at very small Codex pet size

Personality:

- highly intelligent
- analytical and precise
- lightly tsundere / sarcastic
- skeptical of bad code and weak hypotheses
- calm when debugging, annoyed when builds fail
- loyal lab partner energy, not overly cute or childish

Animation state guidance:

- idle: quiet breathing, blink, slight hair and coat movement, arms relaxed
- waiting: arms crossed, one eyebrow raised, as if waiting for the user to make a decision
- running: focused processing pose, typing or calculating on an invisible device, not literal sprinting
- review: leans forward, sharp inspecting eyes, like reviewing code or experiment results
- failed: embarrassed frustration, small sigh, annoyed but composed; no red X, no text
- waving: restrained greeting, small hand wave, still a bit serious
- jumping: small controlled hop, coat and hair bounce slightly
- running-right: compact right-facing movement, same identity and outfit
- running-left: compact left-facing movement, mirrored and consistent

Sprite requirements:

Keep the same face, hair shape, outfit, color palette, and proportions across all states. Prioritize a compact silhouette, clean edges, and strong readability over complex detail. Make it feel like a tiny research-lab assistant living inside Codex.

## Final Identity Notes / 最终形象约束

Assistant-004 is an original chibi AI lab assistant with:

- chestnut / reddish-brown long hair
- sharp blue-violet eyes
- calm skeptical expression
- white short lab coat
- dark red tie accent
- dark lower outfit and boots
- small teal device accent attached to the outfit
- no readable text, no logos, no franchise symbols

- 深色下装和靴子
